Desarrollo del proyecto

Se decidió por elaborar tarjetas memorice en las que se mostraran los geositios. Estas tarjetas no serían una imagen solamente, se planteó utilizar tecnología de realidad aumentada para enriquecer y diversificar las formas en las que se podía trabajar. Estas se trabajaron en la plataforma de unity y vuforia, ademas de recopilarse imagenes satelitales de los volcanes que comprenden el geoparque y se utilizaron para filtrar los geositios. Finalmente se realizaron ilustraciones vectoriales para vincular el marcador con el modelo.

Software

Después de una investigación de diversas herramientas para crear el memorice se decidio utilizar los software de UNITY Y VUFORIA Unity como herramienta de compilación de modelos y vinculacion al marcador de referencia y Vuforia como plataforma de creacion de VuMark(marcadores propios). El proceso se realizó desde la configuración de elementos separados y la vinculación de Modelo 3D con Marcador

Modelo 3D

El Modelo 3D se trabajo con Rhinoceros 7, las vistas satelitales fueron obtenidas desde https://elevationapi.com/ y fueron convertidas en modelos 3D a los cuales se les vínculo la imagen satelital como textura, creando a la vez un cuerpo que permitiera visualizar la malla en Unity.

Next

Marcador

El marcador fue hecho en base a imagenes almacenadas en Google del Geoparque y sus geositios realizando ilustraciones vectoriales para tener la posibilidad de imprimirlos en madera con impresión láser. El marcador se digitaliza y sube a VUFORIA para posteriormente importarlo a UNITY y obtener que la imagen del geositio sea un marcador donde posicionar el Modelo 3D
