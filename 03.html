Posibilidades

Proyecto con posibilidad de continuidad y mejora, base para un memorice geologico aplicable en varios sectores del país, proceso previo al desarrollo de una aplicación que incorpore la realidad aumentada y permita utilizar las tarjetas de manera interactiva en combinación con la visualización en tres dimensiones, Base de proyecto sumada a investigación de la tecnología y aplicación en la educación.

More

Opciones

Posibilidad de desarrollo de la parte grafica para convertirlo en un memorice geologico sin mayor aplicación considerando la incorporación de simbologias e información como tiempo, formación y ubicación del geositio. Opcion de continuidad con aplicacion de visualizacion de geositios a traves de modelos 3D con vista en 360° para acercar los geositios a los residentes. Desarrollo de tarjetas con impresión laser para incorporar el tacto y lograr reconocer rasgos caracteristicos de los geositios.

Conclusiones

La realizacion de este proyecto fue un proceso largo, se atravesaron bastantes pruebas para conseguir encontrar que hacer y como hacerlo, se probaron varios software de realidad aumentada, se exploraron varios ElevationApi para obtener imagenes satelitales, el proyecto puede parecer simple pero es una base solida para el desarrollo de cualquier propuesta que involucre realidad aumentada y la utilizacion de imagenes satelitales de gran escala. Agradecimientos especiales a GABRIELA TASCON por el apoyo durante la selección de posibles geositios para el desarrollo del memorice en caso de interés.
